```

0:06-0:09  Tap a resort — detail page with cams + snow report

0:15-0:18  Back to browse — filter by "Great" conditions
0:18-0:20  End card: "peakcam.io — Free. No login needed."
```

**Recording instructions:**
- Use iPhone screen recording (9:16 portrait)
- Open peakcam.io in Safari, mobile view
- Scroll smoothly, don't rush
- Pick a resort with good snow data (Mammoth, Whistler, or Jackson Hole)
- Music: lo-fi chill or ambient mountain vibes (royalty-free)
