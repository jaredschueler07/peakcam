# PeakCam Design System

Retro ski-poster aesthetic for **PeakCam** — live webcams, snow reports &
trail conditions for 128 North American ski resorts.

## Direction
- **Aesthetic** — 70s/80s resort-poster nostalgia: chunky humanist display
  type, thick strokes, flat color fills, layered topo lines, badge shapes,
  hand-stamped feel.
- **Palette** — Earth + snow. Forest green, bark brown, cream paper, with
  alpenglow persimmon as the only hot accent.
- **Type** — Fraktur-free. Rounded humanist display (Recoleta-feel) + clean
  humanist sans for body + slab mono for data readouts.
- **Voice** — Casual, insider. "Got the goods." "Pow day." "Lift's spinning."

## Files
- `tokens.css` — colors, type, spacing, shadows, radii
- `components.html` — type, color, button, card, badge, nav, form, cam-tile specimens
- `Dashboard.html` — two full-bleed dashboard variations
- `Deck.html` — 6-slide starter deck

## Product vocabulary (from peakcam.io)
- **Resort** — a ski resort with many cams (e.g. Snowbasin, Willamette Pass)
- **Cam** — a live webcam at a resort
- **Base depth** — inches at base
- **Fresh snow** / **24hr** — new accumulation
- **Conditions** — Great / Good / Fair / Poor
- **Filters** — US/CA state, Fresh Snow, Live Cams, Any Condition
